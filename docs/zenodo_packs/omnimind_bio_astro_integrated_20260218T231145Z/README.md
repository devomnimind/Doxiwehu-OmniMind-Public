Este pack consolida artefatos locais reais (datasets, logs e validações) em trilha integrada.
Continuidade explícita da trilha astrofísica-base: DOI `10.5281/zenodo.18681824`.

Conteúdo:
- `data/`: relatórios JSON/MD técnicos (bio, astrofísico, correlação, Qdrant, fechamento local).
